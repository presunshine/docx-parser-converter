Key Points About Lists

    •	Each numbering reference creates an independent list

    •	Same reference continues numbering, different reference restarts

    •	Levels create hierarchical structure (0 = main, 1 = sub, 2 = sub-sub)

    •	Indentation is controlled by left and hanging indent values

    •	Always use LevelFormat constants (BULLET, DECIMAL, UPPER_ROMAN, LOWER_LETTER, etc.)
