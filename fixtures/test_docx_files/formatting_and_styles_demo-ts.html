Line Spacing Values Explained

Line spacing is measured in twentieths of a point (DXA):

• line: 240 = Single spacing (12pt)

• line: 360 = 1.5 line spacing (18pt)

• line: 480 = Double spacing (24pt)

• line: 276 = 1.15 spacing (Word default, 13.8pt)

Line Rule Options

• lineRule: 'auto' - Spacing adjusts automatically based on font size

• lineRule: 'exact' - Fixed spacing regardless of content

• lineRule: 'atLeast' - Minimum spacing, can expand if needed

5. Inline vs Style-based: When to Use Each

Use Inline Formatting When:

• Formatting is unique and won't be repeated

• Making quick, one-off emphasis like making a single word bold or italic

• Creating document-specific formatting that doesn't need consistency

Use Style-based Formatting When:

• Formatting will be reused throughout the document

• You need consistent formatting across multiple elements

• Working with templates or professional documents

• You want easy global updates (change style definition once, affects all instances)

Summary

This document demonstrates both inline formatting (applied directly to text) and style-based formatting (referenced from styles.xml). Both approaches are valid and serve different purposes in document creation.
